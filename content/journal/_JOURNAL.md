# journal/ — Journal-Beiträge

Kurze Notizen zu einzelnen Bildern, Reisen oder Gedanken. Ein Beitrag = eine
Markdown-Datei → `/journal/<slug>`.

```
journal/
└── <slug>.md     Frontmatter (title, date, excerpt?) + Fließtext
```

Schema & Anleitung: [`../_CONTENT.md`](../_CONTENT.md).
Reihenfolge in der Übersicht: rückwärts chronologisch nach `date`.
